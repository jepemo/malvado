Installation

  • Install Love2d
  • Install Malvado:
luarocks install malvado

For the documentation generation, if you download from github, you need LDoc and Penlight .

Run Examples

love examples/helloworld
# or
love examples/demo
# or
love examples/space_asteroids
# or
love examples/bench
